2024 ‚Äî Short-Term Load Demand Forecasting Using Parallel CNN‚ÄìBiLSTM Hybrid

Accurate short-term load forecasting is critical for efficient power plant operation, resource allocation, and grid stability. However, conventional forecasting models struggle to handle the strong non-linearity and real-world variability present in modern load demand data.

This work proposes a novel short-term load demand forecasting framework specifically designed for the Bangladesh power system. The model is trained and evaluated using an extensive six-year dataset that integrates historical load demand, meteorological variables, and Bangladesh-specific calendar information, capturing diverse seasonal and temporal patterns.

A parallel CNN‚ÄìBiLSTM architecture is employed, where the CNN module extracts spatial features from the combined dataset, while the BiLSTM module learns long-term temporal dependencies and short-term fluctuations. The parallel design enables the model to effectively capture complex correlations in load demand time-series data while improving computational efficiency.

Experimental results demonstrate that the proposed hybrid model outperforms traditional forecasting techniques and baseline neural network models. Performance evaluation using Mean Absolute Percentage Error (MAPE) and R¬≤ score confirms superior prediction accuracy, making the approach suitable for practical applications such as capacity planning, load balancing, and resource scheduling in real-world power systems.
