2024 ‚Äî Short-Term Load Demand Forecasting Using Parallel CNN‚ÄìBiLSTM Hybrid

Accurate short-term load forecasting is critical for efficient power plant operation, resource allocation, and grid stability. Conventional forecasting models often struggle with real-world non-linearity and variability in modern load demand signals.

This work proposes a short-term load demand forecasting framework tailored to Bangladesh. The model was trained and evaluated using a six-year dataset combining historical load demand, meteorological factors, and Bangladesh-specific calendar information to capture seasonal and temporal patterns.

A parallel CNN‚ÄìBiLSTM architecture is used: CNN extracts spatial features from the merged dataset, while BiLSTM learns temporal dependencies and short-term fluctuations. Experiments show improved accuracy versus baseline and traditional methods, evaluated using MAPE and R¬≤ metrics.
